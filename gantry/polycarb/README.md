# Panda Deck (Polycarbonate Base Plate)

Base plate for the **Cub-XL** intended for use with non-contact instruments.
Reproduces the deck layout of the original
[PANDA-BEAR](https://github.com/BU-KABlab/PANDA-BEAR) configuration, so any
labware and instrument positions defined for PANDA carry over to a Cub-XL
built on this plate.

## Files

| File | Purpose |
| --- | --- |
| `PandaDeck.step` | Source CAD — edit this if you need to adjust hole patterns or footprint. |
| `PandaDeck.stl` | Mesh export of the same plate (e.g. for slicing or quick previews). |

## Material

Intended to be machined from **polycarbonate** sheet — hence the folder
name. STL is provided for convenience but the STEP is the authoritative
source for fabrication.

## Compatibility

- Deck: **Cub-XL only**
- Instruments: non-contact (no force/contact loading expected on the plate)
