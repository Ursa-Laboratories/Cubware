# ASMI Vernier Force Sensor Mount

3D-printed mount for attaching a Vernier force sensor to the
[ASMI](https://github.com/BU-KABlab/ASMI_new) system.

## Parts

| File | Purpose |
| --- | --- |
| `force_sensor_mount_back.stl` / `.step` | Back half of the sensor housing. |
| `force_sensor_mount_front.stl` / `.step` | Front half of the sensor housing. |
| `force_sensor_mount_cap.stl` / `.step` | Cap that closes the housing once the sensor is seated. |

Print all three; the `.step` files are the source CAD if you need to
modify the design.
